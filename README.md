## **Installation**

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/squid-game-profile.git

2. Navigate to the project directory:
   ```bash
   cd squid-game-profile

3. Install dependencies:
   ```bash
   npm install

4. Run the app locally:
   ```bash
   npm start 

## **Technologies Used**
